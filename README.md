🌊 IoT-Based Flood Prediction System

A real-time flood monitoring and prediction system that combines IoT sensors and Machine Learning to provide early warnings and reduce disaster impact.

🚀 Overview

This project integrates ESP32-based environmental sensing with a Machine Learning model to predict flood risk. Sensor data such as rainfall, water level, temperature, and humidity is transmitted to a backend server, where a trained model analyzes the data and classifies the flood risk as Safe or Danger. A live dashboard provides real-time visualization and monitoring.

🧠 Features

📡 Real-time data acquisition using ESP32

🤖 Machine Learning-based flood prediction (Random Forest)

🌐 REST API built with Flask

📊 Live dashboard using Streamlit

📝 Continuous data logging for analysis

🚨 Early warning system for flood risk
